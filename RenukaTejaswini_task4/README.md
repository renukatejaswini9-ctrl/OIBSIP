# Login Authentication System

## Oasis Infobyte Internship – Task 4

### 📌 Project Description

The Login Authentication System is a simple web application developed using **HTML**, **CSS**, and **JavaScript**. It allows users to create an account, log in with their credentials, access a protected home page, and securely log out. User information is stored using the browser's **Local Storage** for demonstration purposes.

---

## ✨ Features

- 👤 User Registration
- 🔐 User Login Authentication
- 🏠 Protected Home Page
- 🚪 Logout Functionality
- ✅ Form Validation
- 💾 Local Storage Support
- 📱 Responsive User Interface

## 📖 How to Use

1. Open the Registration page and create a new account.
2. Login using the registered email and password.
3. Access the Home page after successful authentication.
4. Click the Logout button to end the session.
5. If a user tries to access the Home page without logging in, they are redirected to the Login page.
