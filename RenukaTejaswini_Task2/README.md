# Tribute Page – Dr. A.P.J. Abdul Kalam

## Oasis Infobyte Internship – Task 2

### Project Description
This is a Tribute Page dedicated to Dr. A.P.J. Abdul Kalam, the former President of India and one of the country's most respected scientists. The webpage highlights his life, achievements, inspirational quotes, and contributions to the nation.

### Features
- Responsive webpage design
- Hero section with title and image
- Inspirational quote
- Biography section
- Major achievements
- "Why I Admire Dr. A.P.J. Abdul Kalam" section
- Simple and attractive layout using HTML and CSS

### Technologies Used
- HTML5
- CSS3
